Escribir un programa que calcule el máximo común divisor (MCD) de dos enteros positivos. Si M >= N una función recursiva para MCD es:\\

MCD = M si N =0\\
MCD = MCD (N, M mod N) si N <> 0\\

En caso de que los valosres ingresados cumplan que M < N el programa debe intercambiar los valores.
